================================================================================
    WHAT HAPPENS WHEN YOU RUN
================================================================================

1. The start script checks for g++ and Python
2. It compiles the C++ compiler from source
3. It installs Python dependencies (Flask)
4. It starts the Flask server on port 5000
5. You can access the dashboard at http://localhost:5000

Behind the scenes:
    - You type an expression
    - JavaScript sends it to Flask API
    - Flask runs the C++ compiler
    - C++ compiler processes the expression
    - Results are sent back as JSON
    - Dashboard visualizes everything!
